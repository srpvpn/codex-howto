# Plan Mode Examples

Real-world examples for Codex plan mode.

## Example 1: API build-out

```text
User: /plan Build a REST API for a blog

Codex:
- identifies schema, auth, tests, and docs work
- groups the implementation into phases
- asks for approval before broad edits
```

## Example 2: Migration planning

```text
User: /plan Migrate from MongoDB to PostgreSQL

Codex:
- proposes dual-write or cutover strategies
- lists rollback points
- flags validation and performance risks
```

## Example 3: Large refactor

```text
User: /plan Convert the React app from class components to hooks

Codex:
- inventories the component set
- suggests migration order
- separates preparation, implementation, and verification
```
