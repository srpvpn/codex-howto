# Codex How-To Brand Assets

Brand assets for the Codex How-To repository.

## Asset groups

- `resources/logos/` for main logo variants
- `resources/icons/` for square icon variants
- `resources/favicons/` for browser and app favicon sizes
- `assets/logo/` for raw mark/wordmark variants

## Notes

- Logos and icons use the `codex-howto-*` naming scheme throughout the repository.
