# Codex How-To Design System

## Identity

The visual mark remains a compass-plus-bracket symbol to represent guided navigation through code.

## Palette

- black: `#000000`
- white: `#FFFFFF`
- gray: `#6B7280`
- accent green: `#22C55E`
- dark background: `#0A0A0A`

## Usage

- use the logo for documentation headers
- use the icon for avatars and square placements
- use favicon variants for browser and app surfaces
