# Documentation Plugin

Example Codex plugin for documentation-oriented skills.

## Suggested Codex packaging

- `.codex-plugin/plugin.json`
- `skills/` for reusable documentation workflows
- optional `.mcp.json` if the plugin needs external docs or repo systems
- optional `.app.json` if the plugin is connected to app surfaces

## Good fit for bundled skills

- generate API docs
- refresh README sections
- validate docs against code changes
- produce examples from current code paths
